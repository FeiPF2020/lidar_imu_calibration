# lidar_imu_calibration
calibrate lidar (airy or livox) and imu(pix,xsense) to get the parameters

# 需要三个package
依次安装下面的四个软件包
- [Livox-SDK](src/livox_ros_driver/Livox-SDK)
使用cmake 安装
- [livox_ros_driver](src/livox_ros_driver)
执行脚本安装
- [LiDAR_IMU_Init](src/LiDAR_IMU_Init)
- [rslidar_sdk](src/rslidar_sdk)
